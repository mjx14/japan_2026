# Proyecto Japan 2026

Este repositorio documenta el viaje a Japón del 4 al 19 de marzo de 2026, desde Bilbao a Tokio, Osaka, Kioto y regreso. Aquí encontrarás:

- Guía e itinerario detallado día a día
- Recursos visuales y mapas
- Enlaces útiles para reservas y compras
- Consejos prácticos para viajeros

## Estructura del proyecto

- `guia/itinerario.md`: Guía principal con el itinerario y actividades
- `resources/images/`: Imágenes representativas de cada actividad y destino
- `README.md`: Explicación general del viaje y del repositorio

## Sobre el viaje

El viaje está pensado para disfrutar de Japón de forma relajada, visitando los imprescindibles de Osaka, Kioto, Nara y Tokio, incluyendo Universal Studios Japan y excursiones recomendadas. El itinerario se irá actualizando y enriqueciendo con experiencias, fotos y consejos.
